# Personal Portfolio Website

This is a professional, responsive personal portfolio website built as part of the Daulkor Technology Internship Project Assignment.

#Features
- Responsive Design: Optimized for Desktop, Tablet, and Mobile views.
- Smooth Animations: Integrated with the AOS (Animate On Scroll) library.
- Interactive UI: Interactive buttons, hover effects, and smooth scrolling.
- Direct Contact: Functional links for Email, LinkedIn, and Phone dialer.

#Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Libraries: FontAwesome (Icons), Animate On Scroll (AOS)
- Deployment: GitHub Pages

#Project Structure
- index.html - The main structure of the site.
- index.css - Custom styling and responsive media queries.
- index.js - AOS initialization and interactive logic.
- images/ - Folder containing all project-related assets and screenshots.
